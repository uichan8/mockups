# Gallery App Mockup - 사진 정리 앱 목업

조이스틱 기반 인터랙션으로 사진을 정리하는 모바일 갤러리 앱 목업입니다.
아이폰 스타일의 프레임 안에서 동작하며, 순수 HTML/CSS/JS로 구현되어 별도 빌드 없이 브라우저에서 바로 실행할 수 있습니다.

## 주요 기능

### 사진 탐색
- **필름스트립** - 상단 썸네일 가로 스크롤로 사진 간 빠른 이동
- **슬라이더** - 좌/현재/우 3장 카드 전환 애니메이션
- **메타 정보** - 각 사진의 촬영 날짜 및 위치 표시

### 조이스틱 제스처
사진 영역을 터치(클릭)하면 조이스틱이 나타나며, 드래그 방향에 따라 다른 동작을 수행합니다.

| 제스처 | 동작 |
|--------|------|
| ← → 좌우 드래그 | 이전/다음 사진으로 이동 |
| ↑ 위로 드래그 | 사진 삭제 |
| ↓ 아래로 드래그 | 선택된 앨범에 추가 |
| 한 번 탭 | 사진 전체화면 보기 |
| 두 번 탭 | 실행 취소 |
| 꾹 누르기 (600ms+) | 즐겨찾기 토글 |

### 앨범 관리
- 하단 앨범 카드 가로 스크롤 (드래그 지원)
- 스크롤 중앙에 가장 가까운 앨범이 자동으로 선택됨
- 앨범 추가 버튼 제공

### 기타
- **다크/라이트 테마 전환** - 상단 바 아이콘으로 토글
- **도움말 팝업** - 제스처 사용법 안내
- **즐겨찾기** - 별표 아이콘 및 팝업 애니메이션

## 파일 구조

```
mockups/
├── index.html                      # 진입점 (photo_organizer.html로 리다이렉트)
└── gallering/
    └── photo_organizer.html        # 메인 애플리케이션 (HTML + CSS + JS 단일 파일)
```

## 실행 방법

별도의 설치나 빌드 과정이 필요 없습니다. `index.html`을 브라우저에서 열면 자동으로 갤러리 앱 목업 화면으로 이동합니다.

```bash
# 방법 1: 파일을 직접 브라우저에서 열기
open index.html

# 방법 2: 로컬 서버 사용
npx serve .
```

## 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - Flexbox, CSS 변수, 애니메이션, backdrop-filter
- **Vanilla JavaScript** - 프레임워크 없이 순수 JS로 구현
